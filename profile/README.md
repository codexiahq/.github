## üëã About Us

**Codexia** is an open-source initiative dedicated to bridging the gap between browsing and coding. We believe developers shouldn't have to switch contexts to test a snippet, debug a layout, or inspect an element. Our mission is to bring the power of a desktop IDE directly into your browser.

## üöÄ Our Projects

### üß© [Codexia Extension](https://github.com/codexia/codexia)
The core browser extension that transforms your browser into a versatile coding workspace.
- **Polyglot Runtime:** Execute JavaScript, TypeScript, Python (via Pyodide), and HTML/CSS instantly.
- **Live DOM Inspector:** A streamlined alternative to DevTools for quick layout debugging.
- **Snippet Manager:** Build your personal library of reusable code snippets.
- **Smart Injection:** Automatically detects code blocks on web pages and makes them runnable.

### üåê [Codexia Web](https://github.com/codexia/codexia-web-app)
The official landing page and documentation platform.
- **Modern Stack:** Built with Next.js 14 (App Router), TypeScript, and Tailwind CSS.
- **Interactive:** Features live demos and comprehensive guides.
- **Performance:** Optimized for speed and accessibility.
